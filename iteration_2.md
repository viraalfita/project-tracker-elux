# Iteration 2

## Kanban Board Drag-and-Drop

- Implemented drag-and-drop for Kanban board cards using `@dnd-kit/core`.
- Task status now updates by dragging a card into a different column.
- Removed the "Move Left" and "Move Right" buttons from all board cards.
- Admin and Member roles can drag cards to change status.
- Manager and Viewer roles are read-only; dragging is disabled.
